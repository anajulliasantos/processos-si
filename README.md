# ⚙️ Mapeamento de Processos Acadêmicos - Sistemas de Informação

O repositório abrange as rotinas acadêmicas realizadas, geralmente, no início do semestre; ressalta-se que os trâmites mapeados podem sofrer adaptações em casos excepcionais previstos no regimento.

## 📌 Sobre o Projeto

Este repositório é fruto de um Trabalho de Conclusão de Curso (TCC) desenvolvido no semestre de **2026.2** para o curso de **Bacharelado em Sistemas de Informação**, **IFBA - Vitória da Conquista**. 

O projeto tem como objetivo principal documentar, modelar e padronizar os fluxos administrativos e acadêmicos do curso. Através da modelagem visual em BPMN, busca-se oferecer uma visão clara das regras de negócio e das transições de responsabilidade entre discentes, docentes, colegiado, coordenação e demais setores da instituição.

⚠️ **Atenção:** Os fluxos aqui documentados refletem o mapeamento atual das rotinas, mas não são estáticos. Eles representam processos orgânicos que necessitam de melhorias constantes e, portanto, podem sofrer alterações e atualizações contínuas para se adequarem à evolução e otimização das operações institucionais.

## 🛠️ Estrutura e Ferramentas

* **Notação:** BPMN 2.0 (Business Process Model and Notation).
* **Documentação:** Cada diagrama acompanha descrições estruturadas detalhando as etapas e responsabilidades de cada raia de atuação (Discente, Setores de Protocolo e Registros Escolares, Coordenação de SI e Colegiado).
* **Arquivos:** Os diagramas visuais encontram-se exportados no formato `.png`.
